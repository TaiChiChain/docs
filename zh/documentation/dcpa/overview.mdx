公链中缺乏隐私保护是阻碍用户，尤其是企业用户使用和构建基于区块链的商业应用的核心因素之一。
为了解决这个问题，Axiomesh 需要为用户提供强大的隐私保护能力。结合设计原则，
Axiomesh 提出了符合隐私的保护协议 DCPA（具有可审计性的去中心化隐私协议）。
DCPA 在整体架构中采用孪生 Token 模型，以确保协议的可组合性和灵活性。它基于最前沿的加密技术，确保交易内容和身份的安全和隐私，
同时确保符合监管要求。在接下来的部分中，我们将介绍我们的 zkToken、zkDID。
