---
title: "验证者"
---

在 AxiomLedger 的 POS 中，只有验证者才有权利出块，并获得出块的 stake 和 gas fee 收益，而节点注册加入到网络后的角色是数据节点，数据节点是没有质押池，也无法成为共识出块节点的，
想要成为验证者节点，需要 Operator 发送一笔加入候选者节点的交易，以创建质押池，当节点的质押池的代币数量满足要求后，则会在下一个 epoch 自动成为验证者节点。

# 创建质押池

创建质押池，需要使用节点 Operator 的私钥发送一笔交易调用 Staking 系统合约创建质押池(同时会 mint 一个 LST 绑定在质押池上以收取抽水奖励)，创建后节点会转变为候选者节点。
operator使用以下命令进行创建：
```shell
./axiom-ledger node join-candidate-set \
--node-id YOUR_NODE_ID \
--commission-rate 1000 \
--sender b6477143e17f889263044f6cf463dc37177ac4526c4c39a7a344198457024a2f
```
需要指定节点id，节点id在节点注册后会自动分配，以及指定抽水比例，抽水比例的base是10000，即0对应0%抽水，10000对应100%抽水，sender必须是operator的私钥。
交易执行确认后，会创建质押池，mint出一个绑定的LST，用于接收收益抽水，后续该节点即可接受委托质押。

# 成为验证者

当节点的质押池中的代币满足要求时，在下一个epoch，节点会自动被选为验证者节点，出块的概率和质押池代币数量对应，代币越多，出块概率越高。
注意当epoch变更时验证者节点的代币数量不满足要求后，则节点会自动转变为候选者节点，失去出块权利，当然，如果后续质押池代币数量满足要求后，可以继续成为验证者节点。

# 验证者奖励

验证者出块时会获取一定的收益，该收益包含两部分：
1. **stake通胀的奖励**：每个epoch会根据全网质押比例计算出一个收益率（质押比例低时收益率高，比例高试收益率低），并计算出epoch内每个块的奖励。
2. **区块的gas fee收益**：当前会将区块的所有gas fee收益分配给节点。

收益不会实时分配，而是在epoch结束时进行分配，并再由质押池分配给委托质押者。

# 节点退出
当节点运营方不想再运营节点后，可以主动退出，但是退出需要满足Staking的限制，节点退出后，质押池会关闭，无法被委托质押，只能解锁和提款。

operator使用以下命令进行创建：
```shell
./axiom-ledger node exit \
--node-id YOUR_NODE_ID \
--sender b6477143e17f889263044f6cf463dc37177ac4526c4c39a7a344198457024a2f
```
需要指定节点id，，sender必须是operator的私钥.
交易执行确认后，质押池会立即关闭，非验证者节点可直接停止节点，但是验证者节点必须延后到下一个epoch停止节点(因为当前epoch还是验证者会出块)。
