## Axiomesh

Axiomesh 是一个专为大规模商业采纳而构建的模块化区块链。它旨在通过加密经济赋能实体经济。为了实现这一目标，Axiomesh提出了一种具有统一流动性的新区块链架构，以降低用户门槛、保护用户隐私并增强可扩展性。

## AxiomLedger

AxiomLedger 是 Axiomesh 网络的共识层，用于资产发行、交易结算和验证服务。整个架构由高度去中心化的复制节点系统和强大的共识算法保障。

## AXM

`AXM`是`AxiomLedger`的原生代币。用户与`AxiomLedger`的交互过程中会需要使用`AXM`。

您可以访问水龙头来获取一些测试代币来与区块链系统进行交互（主网和各测试网水龙头链接可在[资源](/zh/documentation/getting-started/resources/resources)处查找）。

你可以访问[原生代币](/zh/documentation/economics/coin)以获取更多详细信息。
