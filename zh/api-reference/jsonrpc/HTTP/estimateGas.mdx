---
title: "eth_estimateGas"
api: "POST /"
---

生成并返回完成交易所需的gas量估计值。该交易不会添加到区块链中。请注意，由于包括EVM机制和节点性能在内的多种原因，估计值可能远高于交易实际使用的gas量

### Body

  请求标识符
</ParamField>

<ParamField body="jsonrpc" type="string" placeholder="2.0">
  正在使用的JSON-RPC版本
</ParamField>

<ParamField body="method" type="string" placeholder="eth_estimateGas">
  进行调用的远程方法名称
</ParamField>

<ParamField body="params" type="array">
  包含传递给远程方法的参数列表

  <Expandable title="TRANSACTION CALL ARGS">
     <ParamField body="from" type="string">

     </ParamField>

        20 字节 - 交易指向的地址
     </ParamField>

     <ParamField body="gas" type="string">

     <ParamField body="gasPrice" type="string">

     </ParamField>

     <ParamField body="value" type="string">
        (可选)交易携带的值(十六进制)
     </ParamField>

   </Expandable>
</ParamField>
