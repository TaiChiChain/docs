title: "JS SDK"
---

`AxiomLedger` JS SDK分叉自[ethers.js](https://github.com/ethers-io/ethers.js)。因为`AxiomLedger`是一条以太坊等效链，所以以太坊SDK可以无缝对接。未来会支持`Axiom`治理等特殊功能。

## 安装

运行一下命令以安装`AxiomLedger` JS SDK:

``` shell
npm i @axiomesh/axiom
```

## 简单示例

以下是一个简单的使用示例:

```javascript
const ethers = require('@axiomesh/axiom').ethers;

const url = 'testnet-url';
const provider = new ethers.JsonRpcProvider(url);

async function getBlockInfo() {
    // get the block info of genesis block
    const block = await provider.getBlock(1);
    console.log(block);
}

getBlockInfo();
```
