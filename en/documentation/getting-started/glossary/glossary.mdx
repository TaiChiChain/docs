### System Contract
The built-in contract in AxiomLedger, which does not need to be deployed and can be called directly, similar to Ethereum's Precompiled Contract.

### Sparse Merkle Tree
Sparse Merkle Tree (SMT), which has higher read and write performance than traditional Merkle trees.

### Pruning Scheme
Pruning scheme can prune the blockchain ledger history, reduce the amount of disk space that running a node requires, and improve the read and write performance of ledger.

### DCPA
Decentralized Confidential Protocol with Auditability.
