---
title: "eth_chainId"
api: "POST /"
---

返回当前网络的链ID


### Body

<ParamField body="id" type="number" placeholder="1">
  请求标识符
</ParamField>

<ParamField body="jsonrpc" type="string" placeholder="2.0">
  正在使用的JSON-RPC版本
</ParamField>

<ParamField body="method" type="string" placeholder="eth_chainId">
  进行调用的远程方法名称
</ParamField>

<ParamField body="params" type="array">
  包含传递给远程方法的参数列表
</ParamField>

### Response

<ResponseField name="id" type="number">
  返回请求的唯一标识符
</ResponseField>

<ResponseField name="jsonrpc" type="string">
  返回正在使用的JSON-RPC版本
</ResponseField>


<ResponseField name="chainId" type="string">
  以字符串形式返回十六进制值，表示当前链id的整型数
</ResponseField>

<RequestExample>

```bash Example Request
curl  'https://<testnetwork>' \
--header 'Content-Type: application/json' \
--data '{
    "jsonrpc": "2.0",
    "method": "eth_chainId",
    "params": [],
    "id": 1
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
    "jsonrpc": "2.0",
    "id": 1,
    "result": "0x54c"
}
```

</ResponseExample>
