---
title: "JS SDK"
---

`Axiom` JS SDK is forked from [ethers.js](https://github.com/ethers-io/ethers.js). Since `Axiom` is an Ethereum compatible blockchain, so any Ethereum sdk can use seamlessly. In the future, we will add more features such as governance in SDK. 

## Install

run the following command to install axiom sdk

``` shell
npm i @axiomesh/axiom
```
