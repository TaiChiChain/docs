## Axiomesh

Axiomesh是一个开源区块链项目，旨在建立一个与 Web3 行业的可持续增长相一致的金融服务基础设施。Axiomesh构建在一个高度可扩展、
隐私安全以及可组合的区块链账本AxiomLedger之上，为去中心化金融构筑信任、合规的基石。Axiomesh 不只是一条链，其涵盖整个应用生态系统、以港元
和实物资产支持的稳定币以及一个旨在推动 Axiomesh 发展的基金会。

## AXM

`AXM`是链上原生的与港元1:1锚定的稳定币。用户与`AxiomLedger`的交互过程中会需要使用`AXM`。

用户可以访问[这里]()以购买`AXM`。在测试网中，你可以访问[水龙头]()来获取一些测试代币来与区块链系统进行交互。

你可以访问[稳定币](../../economics/stablecoin.zh)以获取更多详细信息。
